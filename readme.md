## O TESTE

 - Criar uma API utilizando todo seu conhecimento em .Net com as seguintes rotas

### Criar usuario
 - POST /api/v1/usuarios

### Cadastro de produto
 - POST /api/v1/produtos

### Buscar Produtos
 - GET /api/v1/produtos

### Deletar Produtos
 - DELETE /api/v1/produtos
